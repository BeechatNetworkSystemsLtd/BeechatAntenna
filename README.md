# Beechat Antenna
## _Open source RF Antenna for 863-928 MHz_

Beechat Antenna is an antenna with a U.FL male connector with a high efficiency.

## Features

- Average 80-90% efficiency
- Max. 10W input power
- Operating frequency of 863-928 MHz
- U.FL male connector
